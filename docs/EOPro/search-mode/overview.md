# Search mode

User doesn't need to be logged-in to use Search Mode. This mode enables users to locate and explore geospatial datasets from both public (e.g., Sentinel-1, Sentinel-2) and private/commercial sources (e.g., PlanetScope and SkySat). It provides an intuitive user interface that includes dataset selection, area of interest (AOI) selection, and date range selection to refine searches.

Users can configure advanced parameters, such as acquisition modes for Sentinel-1 or cloud coverage limits for Sentinel-2, to enhance search precision. Once the search is executed, results are displayed in an interactive list of assets, allowing users to browse, visualize, and inspect individual datasets. The application also offers dynamic footprint highlighting and asset selection to enhance navigation.

# Data source catalogues available through EOPro

EOPro provides access to the geospatial datasets from both public and commercial sources that are available through EODH hub to enable users to perform advanced Earth Observation analyses.

The available datasets include satellite imagery from:

* Sentinel-2 ARD,

The dataset contains modified Copernicus Sentinel-2 (Level 1C data processed into a surface reflectance product using ARCSI software (Level 2))._
